# Byte Core 0.1.0

Byte Core 0.1.0 is the planned first functional self-managed release. This note is included now only to make candidate artifacts reproducible; it does not announce a release or authorize a tag.

The candidate contains the dependency-light lifecycle CLI, canonical deployment templates, exact-plan install/update/removal proofs, optional Bash and Zsh integration, local-first Byte Care diagnostics, and repository-native Codex guidance.

Known limitations:

- support is limited to the host matrix recorded in `docs/support-matrix.md`;
- remote update discovery and automatic diagnostic reporting are unavailable;
- enterprise fleet management, multi-administrator policy, and paid features are outside v0.1;
- elevated installation and package-manager integration are not provided; and
- the release remains blocked until all automated and manual release-gate evidence passes.
